# ROS
### Beginner: CLI Tools
* Utilized `rqt_graph` and `turtlesim` to map node-to-node communication and verify data flow.
* Understanding the theoretical architecture of Nodes, Topics, Services, Parameters, and Actions.
* Using `rqt_console` for logging, launching multiple nodes, and recording/playing back telemetry data.

### Beginner: Client Libraries (Application)
* **Workspace & Package Management:** Creating standard ROS 2 workspaces and initializing new packages.
* **Build Systems:** Using `colcon` to compile packages and manage dependencies.
* **C++ Implementation:** Writing a simple publisher and subscriber using the `rclcpp` library.
* **Python Implementation:** Writing a simple publisher and subscriber using the `rclpy` library
